# API чатов и сообщений 
Тестовое задание для компании **hitalent**

Небольшое api позволяющее создавать чаты и писать в них сообщения

Существующие методы

POST /chats - создать чат

`` { "tittle": "test tittle" } ``

POST /chats/{id}/messages - отправить сообщение в чат

`` { "text": "test message" }``

GET /chats/{id} - получить чат и последние N сообщений
``также можно указать в query запроса ?limit= , что позволит получить N последних сообщений. Стандартное значение limit=20.
limit не может быть больше 100``

DELETE /chats/{id} - удаление чата со всеми его сообщениями

### Запуск

`docker build -t hitalent_tt:local .`

`docker-compose up`
